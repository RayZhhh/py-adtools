# Installation

We recommend using Python 3.10 or newer.

## Install from PyPI

You can install `py-adtools` directly from PyPI using pip:

```shell
pip install py-adtools
```

## Install from Source

If you need the latest version, you can also install it directly from the GitHub repository:

```shell
pip install git+https://github.com/RayZhhh/py-adtools.git
```
